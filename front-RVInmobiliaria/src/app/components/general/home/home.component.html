Resumen

Para poder registrar un inmueble, se necesita tener registrado una ciudad.
Para poder registrar una ciudad, se necesita tener registrado un departamento.
Para poder registrar un departamento, se necesita tener registrado un país.

En la tabla de departamento, no se guarda el nombre ni el código del paise, se guarda el ID.
En la tabla de ciudad, no se guarda el nombre ni el código del departamento, se guarda el ID.
En la tabla de inmueble, no se guarda el nombre ni el código de la ciudad, se guarda el ID.
Los ID's se guardar para hacer una referencia a las otras tablas (Llave foranea) para no tener informacion duplicada en la DB.
